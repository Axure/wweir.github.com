一般连网思考方向

ip link set wlan0(eth0) up

  • 启用dhcp

dhcpcd wlan0(eth0)

  • 更改DNS (更改前先ping -c 3 8.8.8.8 )

echo nameserver >> /etc/resolv.conf

wifi-menu
