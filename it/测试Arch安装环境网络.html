测试Arch安装环境网络

一般连网思考方向

  • 启用接口

ip link set wlan0(eth0) up

  • 启用dhcp

dhcpcd wlan0(eth0)

  • 更改DNS (更改前先ping -c 3 8.8.8.8 )

echo nameserver >> /etc/resolv.conf

  • 加密的无线

wifi-menu

  1. pppoe-setup
  2. name:()
  3. eth0:[eth0,wlan0,……k]
  4. valua:no
  5. server:server
  6. passwd:
  7. repasswd:
  8. firewall:[0,1,2]
