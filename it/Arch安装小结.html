安装yaourt

yaourt使用中无需root权限

  1. vim /etc/pacman.d/mirrors
  2. 加入 
    [archlinuxfr]
SigLevel = Optional TrustAll
 Server = http://repo.archlinux.fr/$arch
  3. pacman -S yajl
  4. pacman -S yaourt
  5. yaourt -Syua
  6. pacman -S yaourt
